# Paper Explainer Skill 示例

本目录包含 paper-explainer skill 的使用示例。

## 示例 1：完整工作流

查看 `full_workflow/` 目录，了解从输入到输出的完整流程。

## 示例 2：输出格式

查看 `output_examples/` 目录，了解生成的报告格式。

## 快速开始

1. 将 `skills/paper-explainer/` 放到你的项目
2. 按照 SKILL.md 的工作流程执行
3. 查看生成的报告

## 示例说明

### full_workflow/

展示从论文输入到最终 HTML 报告的完整流程。

- **input/sample_paper.txt**: 论文文本片段示例（DeepSeek-R1 论文摘要）
- **output/**: 生成的报告和 SVG 插画

### output_examples/

展示不同类型的输出示例。

- **svg_samples/**: SVG 插画示例
- **html_samples/**: HTML 报告示例

## DeepSeek-R1 示例

本示例展示了如何使用 paper-explainer skill 解读 DeepSeek-R1 论文：

1. **输入**：`full_workflow/input/sample_paper.txt` - 论文核心内容摘要
2. **输出**：`full_workflow/output/sample_report.html` - 完整的技术演进复盘报告

这个示例包含了：
- Phase 0-4 的完整解读
- 多个生活化类比
- 训练流程可视化图
- 工程落地手册
